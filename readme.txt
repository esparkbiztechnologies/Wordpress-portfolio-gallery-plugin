== Description ==

  This Plugin is used to display portfolio with YouTube, video or custom embedded code.

- Compatible with any WordPress site.
- Lists portfolio same as WordPress post / page lists.
- Display portfolio lists using by short-code name as "[esb_pg_portfolio_gallery]".
- Also display portfolio using by widget name as "ESB - Portfolio".

== Installation ==

1. Upload the entire esb_portfolio_gallery folder to the /wp-content/plugins/ directory, or just upload the ZIP package via 'Plugins > Add New > Upload' in your WP Admin
2. Activate ESB Portfolio Gallery from the 'Plugins' page in WordPress
